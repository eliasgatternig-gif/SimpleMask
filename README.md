# Simple Mask

**A lightweight tool to mask parts of images directly in the Editor. Works on Mobile & Web!**

Simple Mask solves the problem of "clutter" when creating image occlusions. Instead of generating dozens of new notes for a single image, it lets you draw masks directly onto your *existing* pictures.

It is **non-destructive**: it never modifies your original image file. Instead, it saves the masks as lightweight code hidden behind the image.

### How to Use
1.  **Select an Image:** Click on any image in the "Add" or "Edit" window.
2.  **Draw:** Select the **Pen** tool and drag to cover the text or diagram.
3.  **Review:** On your phone or computer, the answer will be covered. Tap a mask to reveal just that part, or press **Show Answer** to reveal everything.
